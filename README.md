# ElectroHub-Sales-Data-Analysis-
I developed a comprehensive Power BI project named ElectoHub, where I created multiple interactive dashboards and analytical tables to explore and visualize data effectively. The project involved several stages, including data processing, data cleaning, data filtering, data modeling, and data visualization. I utilized DAX (Data Analysis Expressions) for creating calculated columns and measures, and Power Query for data transformation and preprocessing.
# Purpose of the Project
The purpose of the ElectoHub project is to analyze and visualize sales data effectively using Power BI. It aims to transform raw data into meaningful insights by evaluating key performance indicators, identifying sales trends, assessing product performance, and examining the relationship between sales and profit. The project supports data-driven decision-making through interactive dashboards and structured analytical reporting.
# KPI’s Requirement

✅ Total Sales
✅ Total Profit
✅ Total Quantity Sold
✅ Net Sales
✅ Average Discount
✅ Total Number of Orders
✅ Sales Growth Over Time
✅ Profit Margin
✅ Sales vs Profit Relationship

# Business Questions Solved
🔹 Top/Bottom 5 products by Sales / Profit / Quantity Sold

🔹 How sales trends vary over time (Daily, Monthly, Quarterly, Annually)

🔹 Relationship between Sales & Profit

🔹 Compare Sales / Profit / Quantity Sold between any two selected periods

🔹 Average discount offered in each discount category

🔹 Total number of orders

🔹 Order-level details with filters (Product, Date, Customer ID, Promotion Category)

🔹 Sales distribution by different cities
